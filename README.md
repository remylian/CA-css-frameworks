
# My Little App – CSS Frameworks Assignment

A responsive, minimal social media UI built using Tailwind CSS. This project fulfills the CSS Frameworks assignment for Noroff’s FED2 program, showcasing multi-page layout, form validation, theming, and accessibility.

## Features

- Responsive multi-page layout using Tailwind CSS
- Login form with HTML validation and styled error feedback
- Post creation form using <details> for collapsible UX
- Skeleton loaders (animate-pulse) to indicate loading states
- Profile page with avatar, banner, follow button, and post feed
- Accessible dark/light theme toggle across all pages
- Consistent navigation with highlighted active page
- Form feedback styling using red/green rings and dynamic messages

## Technologies Used

- HTML5
- Tailwind CSS 3
- PostCSS
- Prettier (with prettier-plugin-tailwindcss)
- Minimal modular JavaScript (for theme toggle and form logic)

## Prerequisites

Make sure you have the following installed:

- Node.js 
- npm (comes with Node)

## Getting Started

1. **Clone the repository**  
   ```bash
   git clone https://github.com/your-username/CA-css-frameworks.git
   cd CA-css-frameworks
   ```

2. **Install dependencies**  
   ```bash
   npm install
   ```

   Main dependencies:
   - tailwindcss
   - postcss 
   - autoprefixer 
   - prettier 
   - prettier-plugin-tailwindcss 

3. **Start the development server** (watch mode for Tailwind)
   ```bash
   npm run dev
   ```

4. **Open index.html** in your browser manually or use a local server (e.g., Live Server in VS Code).
